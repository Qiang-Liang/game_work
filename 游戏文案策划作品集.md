### 1. 《荒芜情书》— 科幻爱情（2022）

**原作故事概述：**
我在单调、耗竭的实验室与都市生活中，与恋人"小饿"关系走向分崩离析；二人一边在"体贴的惯性"里消耗，一边被一套"去中心化的虚拟轮回世界"所裹挟——记忆可被回读，生命像电影复映，错误与别离被迫重来。小饿知悉自己是"十七天一轮"的"bug"，请求不要被再次唤醒，最终在一段儿歌中消失；我只留下没被拆封的最后情书与"已读"的误发情书，像给荒芜竖的一块墓碑。

**高概念：**
存在的，被遗忘。

**原文节选：**
```
“那时，是2022年，我还尚且保有专业赋予的使命，虽然工作没那么有趣，但很简单，做起来还算尽心尽力。就是得靠音乐解乏，耳机里一直是《杀死那个石家庄人》。

——早晨，拆开一整包五十毫升离心管，取出七根，这就是一次实验所需的量。先去楼上取冰，这些制冰所用的水可以直接拿来饮用，可最终他们被选择只是作为提供低温的容器……

‘……如此生活三十年/直到大厦崩塌……’

——再然后就是将加好的药品溶解摇匀，在涡旋机上震荡的时候你能看见微小的龙卷风在离心管中升起，逐渐演化成摧毁一切家园的狂风，它裹挟乌云，遮蔽骄阳，足够摧毁马孔多使它在这片大地不复出现…… 

‘……云层深处的黑暗/淹没心底的景观/在八角柜台……’

——再然后就是稀释溶液，气泡是最不应该出现在塑料板里的，大的气泡用枪头戳破就好了——说起枪头，谁能想到我有七把小枪（单排）两把大枪（12排），可惜它们都是塑料子弹，气密性也一般，扣动扳机，液体横流，再扣动一下，液体吸收回来，再扣动一下，竟是换弹，扣动扳机意味着换弹，和死亡意味着存在倒有想通的意思……

‘……用一张假钞/买一只假枪/保卫她……’

——小的气泡枪头是戳不破的，再细的枪头，就算拿上绣花针也无济于事……

‘保卫她……’

——离心就好了，离心是最好的祛除小气泡的方法……

‘……她的生活/直到大厦崩塌/夜幕覆盖华北平原/忧伤浸透她的脸……’

——我们俩也离心了，因为无数微小的气泡。”
```

### 2. 《将死之人》— 生命最后的思考（2019）

**原文节选：**
```
他为自己金玉其外，败絮其中的生活忧心如焚，他为自己没有原则的妥协悔恨，他为自己一次又一次不遵从内心的理想，出卖肉体给所谓的高价值而自责。他为自己活在痛苦的汪洋里，即使窒息也绝不上浮的勇气懊恼。而旺盛的表达欲拯救不了他，伟大的词汇拯救不了他，最高的利益拯救不了他，最好的安排同样拯救不了他。
至于爱，所谓的爱。正是他避之不及的危险，更何况拯救？

又很久很久，他把书籍扔在一边，书页再也不能塑造他了，因为爱已将他削砍的面目全非。面目全非的人没有其他欲望，只有停滞不前的欲望。旺盛的表达欲在臭水沟挣扎，爱欲被一只大狗叼去了遥远的村庄，性欲被埋没在情感的乱葬岗。黏腻的才华干枯如荒漠，色彩缤纷的世界成为无声默片。他在平地极目远眺，所见皆是悲剧重叠。他几乎每天早晨降生，每天夜晚悲惨地死去，每一天都是无望的抗争。死亡离他太近太近，所以他从未想过寻死——被爱已没有机会，爱人仍能支撑他。
```

### 3. 《无法到达的信》— 物的书信与人的倒影（2020）

**原作故事概述：**
写给蓝色蜘蛛、手术剪、窗外植物与另一个"无法抵达"的对象的信件，物成为时间、伦理与记忆的镜面。每封信都是一次向自我诚实的练习。

**高概念：**
物的书信折出人的剪影。

**原文节选：**
```
圆头剪先生每天都有事做，二十年，每天都有用到他的地方，从最初我在幼儿园时候的手工，再到小学时候的科技创新项目，当然，圆头剪先生能剪的也只有卡纸和塑料瓶，有时候能碰到泡沫和海绵，它们是圆头剪先生所能接触到的最像结缔组织的事物。尖头剪女士，你羡慕你的先生，你羡慕他不用干脏活累活，可你觉得圆头剪先生充实吗？他虽然没淌过污水也没碰上坚硬的枝干，可他三十年都在做些普通剪刀都游刃有余的事啊，他的价值止步于生活中的繁琐，他与线头、卡纸共存。甚至最近两年，这个家的男主人令你的先生来剪鼻毛。可圆头剪先生，承受了所有的折辱与慢怠，他允许胶水涂在他光亮的皮肤上；他强忍着鼻分泌物粘在他的刀刃上；他目送着自己的价值被湮灭成尘埃。

对不起。我除了说对不起什么也说不出，我没有能力送你们再回到你们梦想中的雪白色的天堂，更不能擅作主张让你们就此退休颐养天年。上次见到你们，令我不禁觉得也许到了我撒手人寰那天，你们仍能满身锋锐，一尘不染。

我曾经总认为，时间是横扫一切的镰刀，可是你们告诉我，有些事物比时间还长久。

我曾经还认为，实现伟大的理想是存在的意义。可是你们告诉我，存在就是存在的意义。
``` 

从前，我一直将见鞍思马理解成两个感情深挚的人之间的默契，直到搬家看见你曾经存在在内的那个黝黑巨大的陶盆，我便突然明白了，原来，“爱”是我流淌在你的血液里，所以我爱你即是爱自己，爱自己便也是爱你。我见到鞍想到马了吗？想到的其实是骑在马上的自己罢了。思念也不是孤独的解药，它只是孤独的容器，眼泪不过是孤独的上清液罢了，一旦眼泪满溢出来，思念就破碎不堪了，眼泪倾泻干净，孤独便也自然流淌。

### 4. 《愚行者的奇行故事》— 文明与权力的史诗寓言（2019） <a id="sec-4"></a>

**原作故事概述：**
以虚构国度"巴吉亚纳"为中心，小说并置多条人物线：学者图留斯被处决与焚书、士兵檀鸿雁被加冕为"唯一将军"却终身负罪、弥诺陶洛斯人的南迁与屠杀、僭主之死与冷清葬礼、开拓者达摩克里斯·西西弗的神话式晚景与无人悼念、伊万·尼禄的童年伤痕，共同拷问"谁书写文明、谁被遗忘"。

**高概念：**
谁书写文明，谁被遗忘。

**原文节选：**
```
军队长驱直入，弥诺陶洛斯人的防线顷刻瓦解。巴吉亚纳的士兵在这场旷日持久却势如破竹的战争中勇猛异常，在多次面对数倍于己的敌军时，他们总能上演以少胜多的戏码。弥诺陶洛斯人的血覆盖在了整座草原上，在先前战争中失去亲人与同胞的巴吉亚纳人将无尽的恨意宣泄在这片土地上，弥诺陶洛斯王在最后的决战中毙命于战场。他们一败涂地，弥诺陶洛斯人离开家乡，被迫向更遥远的南方迁徙，没有人知道他们去向何方，在从今往后的史书中，再没有一处有弥诺陶洛斯人的身影，这个骁勇善战的民族，仿佛一瞬消失在这片大地上。长刀抽出刀鞘、利刃割破脖颈、血液四处飞溅的可怕记忆逼迫着弥诺陶洛斯人在南迁的路上不停跋涉没有尽头。一代又一代弥诺陶洛斯人在永不穷尽的旅途中诞生又死去，死去又诞生。在这之后的漫长岁月里，那些深扎在祖先脑中的梦魇已经渐渐被后人遗忘，弥诺陶洛斯人忘记了血色身影背后阴暗的黄昏，响彻整个黑夜凄厉的哭喊和自己祖母口中不断南下的意义。在最后一个人忘记几百年前的祖先为什么出发后，他们不再南下，驻扎在一片宽广的平原。

在那个决战结束后的夜晚，檀鸿雁站在自己的营帐外看向整个血色的大地，那些年轻的尸体铺在这片贫瘠的土地上。见惯生死的檀鸿雁蓦然深感愧疚，他后悔自己下了那个继续追击的命令。草原夜晚的狂风裹挟着腥肉味冲击着老将军的鼻腔，他弯下腰呕吐，不适感令他晕厥过去。从此他落下了头晕恶心的病根。没有了主帅的命令，杀红了眼的士兵们开始了他们胜利后对战利品的搜刮。没有及时离开的弥诺陶洛斯的可怜的人们，他们遭受着巴吉亚纳的疯狂报复。刚刚失去丈夫的妇女于悲伤中受尽凌辱，再被割断喉咙。单纯的孩童倒在眼泪中，尚在哭泣的婴儿被串连在长矛上。那些俘虏来的士兵被一个一个的砍掉头颅，行刑的刽子手们一共砍坏了一千二百把钢刀。哭泣声与喊杀声只是伴奏，欲望与贪婪才是主角，它们使恨意勾连恨意，像枷锁绑在每个人的心灵上。它们相互索取，肆意增长，从来没有最大值。

在所有的肥料中，战争之后的残骸是最优质的。弥诺陶洛斯人的骨肉使这片贫瘠的草地成为了最肥沃的土地，可是在往后的两千九百年，再没有一个会说话的生灵居住于此。也正是因此，当这片大陆上的其他城市想要寻找这个曾经传播文明的使者——弥诺陶洛斯时，他们不远万里前来，看到的只有不断疯长的茂密森林和零星的爬满藤蔓的碎石块。

从此往后的两千九百八十年，这片大陆上的每个文明都独自生长，没有谁知道彼此的存在，他们曾怀疑过有和自己一样的人们生活在遥远的地平线，可所有的寻找都无功而返。最终，每一个文明都不可置辩的确信，自己是唯一承受时间折磨，没有消亡的文明。直到一个叫西西弗的年轻男人从巴吉亚纳最西边的最隐秘的高塔中走出，不停的往西边前行。当他结束自己环绕大陆的行程，在一个不知名的小村庄隐姓埋名，并收养了一个聪明的孩子时，跟随其后的人们正循着他的旅途探索更宽广的世界。这个每天给儿子绘制地图的父亲并不知道，自己引领了一场伟大又影响深远的浪潮，这场浪潮被后世的人们赋名为开拓运动。

```

### 5.《井中魇》— 乡村恐惧与集体失忆（2019） <a id="sec-5"></a>

**原作故事概述：**
南河改道后的村庄流传"疯女人投井"的故事。夜逃的少年坠入井中，看见"明亮的黑"，传说中的"女鬼"不过是被群体暴力吞没的往事。村民以"正义"压过真相，记忆被修葺成更好讲述的版本。

**高概念：**
被集体正义吞噬的真相，化作一口"人人知其名、无人记其事"的井。

**原文节选：**
```
当他从昏迷中苏醒的时候，他环顾四周，他用力看，他什么也看不见。他已忘却自己是一个看不见的盲人。他索性闭上眼睛，他感受着：四周是狭隘的黑，头顶圆形的窟窿是明亮的黑，他第一次看见黑与黑的差别。他不知道自己身处哪里，除了疼痛感挤压着自己的肌肉，他还感觉明亮的黑挤压着狭隘的黑，狭隘的黑挤压着自己。一层又一层的黑，重叠、缠绕、消解、再生，黑本身并不可怕，在黑中生长的狰狞面孔才恐怖。黑色的恐惧令他联想到那口只存在于人们口中的深井，一产生这层联想，他更加害怕了。深井本身不会令人冷汗直流，深井中那个从浓痰里生长出的、披散白发、白发里寄生着无数虱虫的女鬼是他不断战栗的源流。

```

### 6. 《养殖场与农科院》— 反乌托邦寓言（2020） <a id="sec-6"></a>

**原作故事概述：**
现代化养殖场在农科院的配方饲料加持下高效运转。为保证利润与稳定，场主与科研员不断更迭配方、处理"问题个体"，动物被温柔而彻底地工具化，连"悲鸣"都被制度吸音，真实与良心逐渐被"增产"的话语吞没。

**高概念：**
当饲料与利润控制一切，"被驯化者"与"驯化者"一并沉沦。

**原文节选：**
```
猪肉大王收到饲料后，便嘱咐属下投喂猪群，猪群对这三种猪饲料反响不一，猪仔偏爱A，许是其中糖类较多；发情期的种猪倒是对蛋白质含量高的B情有独钟；即将送入屠宰场的猪却偏偏爱C；可是那些老猪却对这些新饲料不感兴趣，它们年幼时吃过河边草草煮熟的猪菜，青年时也为新鲜蔬菜的美味口感而歌颂新主人的伟大，壮年时因为劣质饲料病死过孩子，死过同伴，如今尝过光荣、熬过苦难日子的老猪们，倒甘愿回到那个食不果腹的幼年，尝一尝河边随处可采的猪菜。他们后来大多都因为病痛被送入了养殖场里的焚尸炉。当然，这都是不值一提的后话了。

```

### 7. 《洗澡间的举报》— 校园微权力博弈（2019） <a id="sec-7"></a>

**原作故事概述：**
宿舍区深夜洗澡扰民，主人公通过正式渠道"报修/举报"解决，导致"便利口子"被堵。原本依赖夜间洗澡的同学不满，文明秩序与个体舒适的张力爆发。主角也开始怀疑自己的动机与方式。

**高概念：**
一则"报修"改变晚间洗澡秩序，文明与便利的冲突在走廊里回响。

**原文节选：**
```
第二天，他病了。有些发烧。兴许是许久没有睡好的缘故。他在床上躺了一上午，中午去开水器那接开水时，当他扭转握把的一瞬，成群的苍蝇像沸腾的水蒸气一般冲向他的眼睛。他过去早已经习惯了，这里本不准放什么东西，可自从第一个人放了，就无可救药的传递下去了。那里自从被人好心地放了一个垃圾桶，就彻底成了整层楼的垃圾箱。人们在那里吃泡面，不，就算不在那吃泡面，都会把餐余垃圾扔在那，调料包、酸奶袋子、泡面盒……不过他们倒是摆得很整齐，一个泡面盒叠着另一个，最后上面再放一个烂苹果。简直像是精心制作的生日蛋糕。可今天他头晕脑胀，看到这些恶心的生物向他冲来就用力地挥舞左手。他的右手便也跟随着左手一起起伏，开水便也没有浇进他的杯子而是铺洒在他的右手上。火热的疼痛令他头脑清醒，他下意识地甩开杯子，杯子从开水器的边缘滚下来，口朝下直直地插进一个泡面盒里，杯口正好覆盖了一个被苍蝇啃食一半的苹果核。

```

### 8. 《漫长的寒冬》— 生命最后的选择（2019） <a id="sec-8"></a>

**原作故事概述：**
被宣判"只剩死亡一条路"的人，决定把剩余时间还给自己：与至亲争执，与自己和解，试着让"体面与理解"成为终点的主题词。他第一次认真练习"选择"。

**高概念：**
当确知死亡将至，他第一次尝试"自己做主"。

**原文节选：**
```
所幸，还有这么最后一个选择：病无所谓了，不再治了。他知道自己的病是个无底洞，他治病的一年，家里已快被挖空了，他不忍心看自己的母亲孤苦无依还生活窘迫。他为自己人生中的最后一个重要的决定牢牢地握在自己手上而欣喜不已。他握着出院单的时候，仿佛握着自己最引以为豪的奖杯，尽管生命在他的手心倏忽流逝。

他每一分每一秒都能感觉到自己的器官在悄悄衰竭，可出院后的每一天对他来说都如同重生，身心轻松如同脱网之鱼。虽然现在正值隆冬，可前些天夜里落下今年初雪的时候，宿舍里的人们都跑出来，在已熄了灯的操场上玩雪的时候，虽然他只能远远的举着伞看，可这一切令他想起从前与伙伴们一起玩耍的时光。雪花踩在脚下，发出“咯吱”“咯吱”的声音，他终究还是没忍住抬手摸了摸，刺骨一如曾经，奇怪的是，他突然觉得一股温暖涌上心头。

在他所认为的自己生命的最后时段里，他的心智似乎也在极速衰老，比起曾经他最喜欢的美人迟暮、英雄落幕的悲剧故事，如今他更愿意去看失散多年的亲人重逢，也更希望人人都过上更完满更幸福的一生。

熟悉的凛冽寒风将他从思绪中驱赶出来。真冷啊。他不禁感慨。他不知道自己还能不能度过这个冬天，今年的冬天跟往年的每一个冬天一样，即寒冷又漫长，可这又怎么样呢？他爱。他爱这种寒冷，他爱这种漫长。

他将自己的毛帽正了正，又继续往前走。

这条充满积雪的路，他走了许久，也想了许多过去。一路走来，他掉下三百八十四根头发，他没数，地母替他数了。

现在他快走到了，临进门时，他将帽子摘下来，抖了抖。便进门去了。

```

“行了行了，我下次就会提前告诉你或者摆在桌子上。”
“对啊，那说明我说了这么多有用，你是在我说了这么多才这么想的。”男人的嗓音嘶哑了。

“我都说了是我的错了，你能不能别说了？”女人把衣服放到了浴室里放干净衣服的架子上。
“这是不同的事，我说的是另一件事。”

“可我得说。你听我说。”

“好了，我要洗澡了。”
他拦了一下门，但门还是关上了。他看向浴室门外洗漱台上嵌入的镜子，那里是一片阴影。又有一个黑点一闪而过，好像是果蝇，它看得到我。毋需质疑。

### 11. 《木流牛马》— 名字里的羞辱与成长（2022） <a id="sec-11"></a>

**原作故事概述：**
一次读错与口音偏差变成长期耻笑的标签。同学把它做成梗，主角把羞耻穿在身上长大，直到通过舞台/社团与自己和解，学会把"标签"变回自己的名字。

**原文节选：**
```

——“诺曼底”号上有二十八名船员，一名女服务员，三十一名乘客，其中十二名是妇女。”就快轮到我了，我想。

——“震荡可怕极了。”下一个，下一个就是我了！

```

### 12. 《伟大的笑话》（梁戕） <a id="sec-12"></a>

**一句话定位：**
一位年迈教授与学生领袖"红毛"，在一场从热血到失语的运动里互为镜像；鲜血在广场铺成"迷宫"，被当局改造成城市"氧吧"，记忆却以草的形态继续生长。

**高概念：**
历史被改写，但记忆以草的形态继续生长。

**原文节选：**
```
乌云满天，狂风无法驱离乌云，反而使其愈发厚重；大湖将要决堤，泥沙堵不住缺口，反而成为使其更为汹涌的帮凶。

星点般的棍棒浇不灭学生的怒火，反而使其怒火中烧。随着行动的扩大化，学生运动席卷F国各地区，参与游行的队伍也不再全部由学生构成，工人、教师、各行各业的劳动者成为庞大行伍的一员。“红毛”成为当之无愧的运动领导者。他号召人民群众在广场和街头建起街垒。

“经济发展了，文化还是老样子，物质消费无止境的进步了，而我们这代人成为了牺牲品。”

“红毛”以他惊人的影响力宣读着他为自己的运动勾画出的纲领。作为一个鼓吹革命的家伙，他成功地不能再成功了。他想出的口号总是一针见血的指出了现今政府的弊病。而对于揭露问题后如何重建新的价值观，他显得有些茫然。

“不要改变雇主，而要改变生活的被雇佣。”

可是对于如何改变的方法论，他不想去上个世纪的哲学家那里寻找答案，基于现实去分析具体而微的理论又不太现实。他的运动也渐渐滑入了只有口号和游行的深渊。至于紧随其后的新世界的纲领，他脑海里只有模模糊糊的几条理想主义的概念，旧的,异化的秩序触之即溃，可是新的秩序却匍匐在地，难以直立。

所以所有的一切都凝聚成了人人心中都有，却完全不同的对现实的愿望，他们的口号变成了“我持有对现实的愿望，因为我相信我的欲望的实现。”这场运动的理念也开始具有一般宗教的概念了——相信。

相信是美妙绝伦的，可相信后该如何做？

“红毛”回答不出，其他运动骨干回答不出，这场运动里每一个义愤填膺的人都回答不出。

```

### 13. 《南河》（陈柯） <a id="sec-13"></a>

**一句话定位：**
一次"改种大豆"的暴富幻觉引发供需崩盘，村民将怨恨转嫁给一位已故老先生与其女儿，除夕夜烧棺施暴，女人投井；南河从此"幸福地生活着"。

**高概念：**
集体暴力的真相被时间修葺成更好讲述的版本。

**原文节选：**
```

于是那年除夕，种豆的和不种豆的人气势汹汹的聚在她的屋子外，辱骂他的父亲是个死了都害人的恶鬼，侮辱她是个得了怪病的灾星。仅仅辱骂是不够的，狂欢是不会止于言语的，他们撬开门锁，撞开房门，将放在屋梁上的棺材扯下来。一些人抱着木柴，一些人在生火，他们要把棺材烧了，这样自己身上的降头就能消了。

那个可怜的女人听到了动静，连忙出来查看。她大声喊叫，趴在自己父亲的棺材上，刚趴下就被人拉开，她挣脱几个男人的钳制，再又趴在那棺材上，没几秒钟，又被拉扯开。就这样来来回回了许多次。她一滴泪也没流。她嗓子喊哑了，就拿地上的木棍打人。可迎接她的是许多木棍。人们早就把她当作病疯子，自然不将她当人看，更何况他们手里是正义的刀，头上戴的是正义的帽子，脖子上挂着的也是正义的牌子。他们烧恶鬼，打疯子有什么错呢？这是对的，对极了。

行凶的男人们与女人们相互致意，仿佛一起合作了一项伟大事业。他们将遍体鳞伤的她和一堆焦土扔在一起便扬长而去。
她目睹着自己的父亲连同棺材被烧成了黑烟。她的衣服早已不能称为衣服，她的皮肤也已经不能称作皮肤，而更像是一个颜料盘。满地都是被扯掉的头发。可她还是一滴泪没掉。肃杀的空气将她伤口处的血液冻成了冰渣，桑树的绿叶也早已经变成黄色调完了。她看向那一滩焦土，又想起了父亲临死前说的话：不说不做不听不看不管，脏水就泼不到你的头上。

她突然笑了，她笑得美极了。

可笑着笑着，她的泪便也涌了出来。

她踉踉跄跄地站起来，转过身向那口井走去，她没回头看自己已经成灰的父亲，也决不再想回望肮脏的南河。没有一点留恋。

"扑通。"

```
