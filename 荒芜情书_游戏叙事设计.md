# 《荒芜情书》游戏叙事设计

## 🎮 游戏概念
**类型：** 情感叙事游戏 + 时间循环机制  
**核心玩法：** 17天循环 + 记忆保存 + 情感选择  
**主题：** 都市情感、自我消磨、虚拟轮回

---

## 🌊 叙事结构设计

### 核心叙事机制

#### 1. 时间循环系统
```
[循环周期：17天]
- 第1-5天：关系建立期
- 第6-10天：关系发展期  
- 第11-15天：关系危机期
- 第16-17天：关系终结期

[循环重置]
- 每次循环结束后，时间回到第1天
- 玩家保留部分记忆和情感
- 可以选择不同的行动路径
```

#### 2. 记忆保存机制
```
[记忆类型]
- 情感记忆：对"小饿"的感情
- 事件记忆：重要事件的发生
- 选择记忆：之前的决策结果
- 真相记忆：关于虚拟世界的认知

[记忆等级]
- 模糊记忆：只能保留大概印象
- 清晰记忆：能记住具体细节
- 深刻记忆：完全保留，无法遗忘
```

#### 3. 情感系统
```
[情感状态]
- 爱意：对"小饿"的深情
- 疲惫：对循环生活的厌倦
- 绝望：对无法改变的无力感
- 希望：对突破循环的渴望

[情感影响]
- 不同情感状态影响对话选择
- 情感状态影响剧情走向
- 情感状态影响结局解锁
```

## 📖 分支剧情设计

### 主要分支线

#### 分支A：接受循环，享受当下
```
[第1循环：初次体验]
- 选择：完全沉浸在恋爱中
- 结果：体验美好的17天
- 情感：纯真、快乐

[第2循环：开始怀疑]
- 选择：开始注意到重复
- 结果：试图改变但失败
- 情感：困惑、不安

[第3循环：尝试改变]
- 选择：主动寻找真相
- 结果：发现"小饿"的秘密
- 情感：震惊、痛苦
```

#### 分支B：拒绝循环，寻求真相
```
[第1循环：保持距离]
- 选择：不与"小饿"深入交往
- 结果：孤独但清醒
- 情感：理智、疏离

[第2循环：探索世界]
- 选择：调查虚拟世界的真相
- 结果：发现更多秘密
- 情感：恐惧、愤怒

[第3循环：寻找出路]
- 选择：寻找打破循环的方法
- 结果：面临艰难选择
- 情感：绝望、决心
```

#### 分支C：拯救"小饿"，牺牲自己
```
[第1循环：发现真相]
- 选择：理解"小饿"的痛苦
- 结果：决定帮助她
- 情感：同情、爱意

[第2循环：制定计划]
- 选择：寻找拯救"小饿"的方法
- 结果：发现拯救的代价
- 情感：犹豫、坚定

[第3循环：最终选择]
- 选择：牺牲自己拯救"小饿"
- 结果：打破循环，但失去一切
- 情感：悲壮、解脱
```

## 💬 对话系统设计

### 对话风格指导

#### 主角（"我"）对话示例
```
[初次见面]
"你好，我是...等等，我们是不是在哪里见过？"
[内心独白] 这种熟悉感很奇怪，就像...就像重复了无数次的梦。

[关系发展]
"小饿，有时候我觉得我们就像在演一场已经演过很多次的戏。"
[内心独白] 但她总是用那种纯真的眼神看着我，让我忘记所有的怀疑。

[发现真相]
"小饿，告诉我真相。我们是不是被困在什么里面了？"
[内心独白] 她的沉默比任何回答都更令人心碎。

[最终选择]
"如果这是最后一次，我想对你说...我爱你，不管这是不是真的。"
[内心独白] 也许爱本身就是真的，即使世界是假的。
```

#### "小饿"对话示例
```
[初次见面]
"你好！我是小饿，因为总是饿所以叫这个名字。"
[内心独白] 她的笑容那么纯真，但眼神深处似乎藏着什么。

[关系发展]
"你知道吗？和你在一起的时候，我感觉自己真的活着。"
[内心独白] 但她的声音里有一种说不出的悲伤。

[揭示真相]
"我...我不能说。如果我说了，一切都会结束。"
[内心独白] 她的眼泪比语言更真实。

[最终告别]
"谢谢你，谢谢你让我体验了真正的爱。现在，让我安静地消失吧。"
[内心独白] 她的声音越来越轻，就像要随风飘散。
```

### 对话选择系统

#### 情感选择
```
[面对"小饿"的纯真]
选项A："我也很开心能遇见你"
→ 情感：爱意+1，怀疑-1
→ 结果：关系更亲密，但循环继续

选项B："我们是不是见过？"
→ 情感：怀疑+1，爱意-1  
→ 结果：关系紧张，但可能发现真相

选项C："不管怎样，我想保护你"
→ 情感：保护欲+1，爱意+1
→ 结果：触发拯救分支

[面对循环的真相]
选项A："我不在乎真假，我只在乎你"
→ 情感：接受+1，爱意+1
→ 结果：接受循环，享受当下

选项B："我一定要找到真相"
→ 情感：决心+1，怀疑+1
→ 结果：拒绝循环，寻求真相

选项C："我们一起逃离这里"
→ 情感：希望+1，爱意+1
→ 结果：尝试共同突破
```

#### 行动选择
```
[实验室生活]
选项A：认真工作，忽略异常
→ 结果：维持现状，循环继续
→ 情感：麻木+1，爱意-1

选项B：调查异常现象
→ 结果：发现更多线索
→ 情感：好奇+1，恐惧+1

选项C：专注于与"小饿"的关系
→ 结果：关系加深，但忽略真相
→ 情感：爱意+1，理智-1

[面对"小饿"的消失]
选项A：接受现实，继续生活
→ 结果：循环重置，但保留记忆
→ 情感：绝望+1，麻木+1

选项B：寻找"小饿"的踪迹
→ 结果：发现更多关于虚拟世界的真相
→ 情感：决心+1，恐惧+1

选项C：试图打破循环
→ 结果：面临艰难选择
→ 情感：希望+1，绝望+1
```

## 📝 文案创作指导

### 创作原则

#### 1. 情感真实
- 表达真实的情感体验
- 避免矫揉造作的表达
- 注重情感的细腻变化

#### 2. 哲学深度
- 探讨存在的意义
- 思考真实与虚幻
- 反思爱情的本质

#### 3. 文学性
- 保持诗意的表达
- 运用丰富的修辞
- 注重语言的节奏

### 文案类型

#### 1. 内心独白
```
[主角独白]
"有时候我觉得自己就像实验室里的实验品，被观察、被记录、被分析。但当她出现的时候，一切都变得不一样了。她让我感觉自己真的活着，真的存在。"

[小饿独白]
"我知道自己不是真的，但我的感情是真的。我不想伤害他，但我也不想消失。如果这是最后一次，我想好好珍惜每一刻。"
```

#### 2. 环境描述
```
[实验室]
```

```
[支线任务：保护小饿]
"你发现了小饿的秘密，知道她可能不是真的。但你的感情是真的，你需要保护她，即使这意味着接受虚假的现实。"

"你找到了打破循环的方法，但这需要付出巨大的代价。你需要做出选择：是接受循环，还是打破它？"
```
